📦 ¿Qué es una skill y cómo se instala?

// Paso 1 — Instala el CLI

Solo necesitas Node.js. El comando npx skills no requiere instalación global.

npx skills add <repositorio> --skill <nombre>

// Paso 2 — Elige e instala

Cada tarjeta tiene su comando listo para copiar. Ejecútalo en la raíz de tu proyecto. El skill se guarda en la carpeta .skills/ y el agente lo lee automáticamente.

// Paso 3 — Compatible con cualquier agente

Claude Code, Cursor, Windsurf, Copilot, Codex CLI… Cualquier agente que lea archivos SKILL.md los detecta y aplica sin configuración extra.

// Paso 4 — Gestión

npx skills list  ·  npx skills update  ·  npx skills remove <nombre>
